## About

Ratify is a simple but powerful JavaScript library for front-end form
validation.

You can easily:

+ Register your own custom validators
+ Customize how validation messages are displayed
+ More to come

## Usage

Ratify requires either jQuery or Zepto.

`$("#myForm").ratify();`
